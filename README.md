## 🚀 Featured Projects

### 🔬 [PE-Org-AI-R: AI Readiness Scoring Platform](https://github.com/BigDataIA-Sat-Spring26-Team-2/BigDataIA-Sat-Spring26-Team-2_CS5)

> *How AI-ready are public companies — really?* This platform answers that question with evidence from 90+ SEC filings, hiring data, and patent records.

**What it does:** Scores 13 public companies across 7 weighted AI-readiness dimensions (infrastructure, talent, governance, R&D, etc.) using a statistical engine with confidence intervals, CV penalties, and risk-adjusted aggregation.

**How it's built:**

| Layer | Details |
|---|---|
| **Ingestion** | FastAPI microservices · Snowflake · AWS S3 · Redis · 1,000+ document sections parsed |
| **Scoring Engine** | Statistical model (Vr/Hr/Org-AI-R) · confidence intervals · CV penalties · risk-adjusted aggregation |
| **RAG Search** | ChromaDB hybrid retrieval (dense + BM25 + RRF fusion) · HyDE query expansion · LLM-generated justifications |
| **Agentic Layer** | LangGraph multi-agent (Supervisor + 4 specialist agents) · HITL decision gate · MCP server · Prometheus observability |
| **Output** | Streamlit portfolio dashboard · auto-generated IC Memos · LP Letters |

`Python` `FastAPI` `Snowflake` `LangGraph` `LangChain` `ChromaDB` `Docker` `Claude API` `OpenAI API` `Redis` `AWS S3` `MCP`
